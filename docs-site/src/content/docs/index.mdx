## Simbología usada en el manual

- **⭐ Ideas clave y buenas prácticas** — conceptos que conviene memorizar o aplicar siempre.
- **⚠️ Errores comunes** — fallos frecuentes al aprender o usar Terraform, y cómo evitarlos.
- **🧪 Laboratorio** — ejercicio práctico para reforzar lo aprendido en cada lección.
- **❓ Preguntas de repaso** — preguntas para autoevaluarte antes de avanzar.
- **🔄 Actualización** — puntos donde el contenido actual difiere de lo enseñado originalmente en el curso, verificado contra la documentación oficial.
